# Cattle Feed Billing System (Basic)

A simple web-based cattle feed billing system developed using PHP and MySQL.  
This project helps cattle feed shop owners manage customers, feed products, purchases, payments, and view ledger reports.

## Features

- Add Customer
- Add Feed Products
- Record Feed Purchases
- Add Customer Payments
- Ledger Report to track customer balance
- Admin Panel for managing all records

## Admin Panel Options

The system includes a basic admin dashboard with the following functions:

- Add Customer
- Add Feed
- Add Purchase
- Add Payment
- Ledger Report
- Logout

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
